<p>And so the only thing you need to know about Excel's AutoRecover is this: the default path for Excel is:</p>
<blockquote><p><code>C:/Users/you/Application Data/Microsoft/Excel</code></p></blockquote>
<p>Application Data is usually a hidden folder, so you might not see it. It is still there.</p>
